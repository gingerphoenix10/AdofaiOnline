# ADOFAI Online
This is an experimental plugin made for [A Dance of Fire and Ice](https://store.steampowered.com/app/977950/A_Dance_of_Fire_and_Ice/) by [7th Beat Games](https://github.com/7thbeatgames/) that aims to add full online support to the game's Local Multiplayer mode through [Steam Networking](https://partner.steamgames.com/doc/features/multiplayer/networking)

Notes:
- The mod is still in very early development, with messy code and many bugs. These will (hopefully) be sorted with time
- Requires [BepInEx](https://github.com/bepinex/bepinex) or [UnityModManager](https://github.com/newman55/unity-mod-manager) set to Assembly mode. DoorstopProxy will cause crashes on the latest version of ADOFAI
